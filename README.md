# Clinica Router AI
Advanced agentic medical routing system using LangGraph and FastAPI.

## Deployment
This space runs a FastAPI server inside a Docker container.

### API Endpoints
- `GET /a2a`: Agent Discovery (Card)
- `POST /a2a`: JSON-RPC 2.0 A2A Protocol Handler
- `GET /health`: Health Check
